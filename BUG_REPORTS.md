#  Баг-репорты (Bug Reports)

### BUG-01: Параметр `theme` не добавляется в сгенерированный код при выборе «Красной» темы
* **Серьезность (Severity):** Critical  
* **Приоритет (Priority):** High  
* **Окружение:** Google Chrome, Desktop  
* **Описание:** При выборе цветовой темы «Красная» в конфигураторе (Шаг 4), параметр темы не добавляется в атрибут `src` итогового `<iframe>`. При этом для других тем (например, «Синяя») параметр добавляется корректно (`theme=blue`). Это приводит к тому, что виджет на стороне клиента загрузится с темой по умолчанию, игнорируя выбор пользователя.
* **Шаги воспроизведения:**
  1. Открыть страницу https://dev.3s.info/eventswidget/
  2. В настройках виджета (Шаг 4) выбрать «Красную» тему (Светлая тема).
  3. Нажать кнопку «Сгенерировать превью».
  4. Скопировать и проверить сгенерированный HTML-код.
  5. *Для сравнения:* Выбрать «Синюю» тему и повторить генерацию.
* **Ожидаемый результат:** В атрибуте `src` iframe присутствует параметр темы, аналогичный другим темам (например, `...&theme=red`).
* **Фактический результат:** Для красной темы параметр `theme` полностью отсутствует в URL. 
  *Код для красной темы:* `<iframe ... src="https://dev.3s.info/widget-active-events/?event_type=2,3,4&event_country=au&event_lang=ru" ...>`
  *Код для синей темы (для сравнения):* `<iframe ... src="https://dev.3s.info/widget-active-events/?theme=blue&event_type=2,3,4&event_country=au&event_lang=ru" ...>`
* **Вложение:**

### BUG-02: Окно превью не обновляется и отображает некорректную цветовую тему
* **Серьезность (Severity):** Major  
* **Приоритет (Priority):** High  
* **Окружение:** Google Chrome, Desktop  
* **Описание:** Окно предпросмотра (превью) виджета не реагирует на изменения параметров в конфигураторе. При выборе «Светлой красной» темы превью отрисовывается в дефолтной «Темно-синей» теме. Кроме того, в превью постоянно отображается сообщение «Нет событий», независимо от выбранных стран и тематик.
* **Шаги воспроизведения:**
  1. Открыть страницу https://dev.3s.info/eventswidget/
  2. В Шаге 4 выбрать «Светлую красную» тему.
  3. Нажать кнопку «Сгенерировать превью».
  4. Обратить внимание на окно превью справа.
* **Ожидаемый результат:** Окно превью обновляется, отображая виджет в выбранной цветовой теме.
* **Фактический результат:** Превью не обновляется, отображается темная тема.

### BUG-03: Ошибки Mixed Content из-за загрузки изображений по HTTP на HTTPS-странице
* **Серьезность (Severity):** Minor  
* **Приоритет (Priority):** Low  
* **Окружение:** Google Chrome, Desktop  
* **Описание:** Страница загружается по защищенному протоколу HTTPS, однако в HTML-коде страницы жестко прописаны ссылки на изображения по незащищенному протоколу HTTP. Браузер автоматически обновляет запросы до HTTPS, но это генерирует множественные предупреждения в консоли.
* **Шаги воспроизведения:**
  1. Открыть страницу https://dev.3s.info/eventswidget/
  2. Открыть DevTools (F12) -> вкладка Console.
* **Ожидаемый результат:** Все ресурсы запрашиваются по протоколу HTTPS. Консоль чиста.
* **Фактический результат:** В консоли отображаются ошибки: `Mixed Content: The page at 'https://...' was loaded over HTTPS, but requested an insecure element 'http://dev.3s.info/uploads/media/.../*.jpg'`.
