## 📊 Краткое резюме (Summary)
В ходе исследовательского тестирования конфигуратора виджета было выявлено **3 дефекта** (1 Critical, 1 Major, 1 Minor). 
Главная проблема заключается в том, что конфигуратор некорректно генерирует код для «Красной» темы (параметр `theme` полностью отсутствует в итоговом URL), а окно предпросмотра не обновляется в реальном времени.

**Окружение тестирования:**
- **OS:** Windows 11 / macOS
- **Browser:** Google Chrome (Latest) + DevTools
- **Tools:** Postman, Charles Proxy (для анализа запросов)
